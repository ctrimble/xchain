Welcome to the XChain 5 Minute Introduction Tutorial
Welcome to the Introduction tutorial. So far you have seen how to create, build and deploy the XChains tutorial project.
One of the features XChains supports is live monitoring. Any changes you make to your xml files will be picked up immediately, saving you the hassle of rebuilding and redeploying your app during development.
Now lets see an example of live monitoring.
Open the introduction index.xhtml in the src/main/webapp/intro/ directory.
Here you see the xhtml source for this page. Towards the end of the file you should see this block of code:
<!--
Congratulations, you have complete the XChains 5 minute introduction tutorial. Return to the root index to see more tutorials.
-->
Remove the comment tags (<!--, -->) surrounding this block and save the file.
Now refresh this page.
