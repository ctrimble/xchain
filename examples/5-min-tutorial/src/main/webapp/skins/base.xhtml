Skinning 5 Minute Tutorial

Step 2

Alright you've just made the index.xhtml page extend the base.xhtml file that also lives in the skin directory.

<?skin system-id="resource://servlet-context/skins/base.xhtml"?>

The skin processing instruction tells XChain's what it needs to complete itself. You've probably noticed that Step 1 disappeared. Go open base.xhtml and uncomment the following line

<!--
<skin:block id="first"/>
-->

Step 1 should be back now. This happened because you were extending the base.xhtml file which didn't know where <div id="first"> should go. The line you uncommented had the matching ID so the skinner put the DIV tag in index.xhtml where the skin:block tag was in the base.xhtml file.

Quick note on the skin namespace, the last step the skinner does is remove anything with the skin namespace. In this case <skin:block id="first"/> is a placeholder so "Step 1" will be put before "Step 2" in the final output.
