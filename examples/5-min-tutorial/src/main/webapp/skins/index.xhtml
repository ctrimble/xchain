Skinning 5 Minute Tutorial

Step 1

Welcome to the skinning 5-minute tutorial. We'll walk through step by step and get you up and rolling with XChain's skinning.

By now you've gone through a few of the other tutorials and so probably know how to work with some of the features that XChain's provides. Since you're familiar with how the files work lets jump right in. Open up the skins/index.xhtml file.

You should see the code below at the top of the document.

<!--
<?skin system-id="resource://servlet-context/skins/base.xhtml"?>
-->

Remove the comment and reload this page.
