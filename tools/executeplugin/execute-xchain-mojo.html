execute:execute-xchain

Full name:

org.xchain.tools:executeplugin:0.5.0-SNAPSHOT:execute-xchain

Description:

Configure Hibernate. Run an XChain.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: runtime.
